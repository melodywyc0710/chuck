Why Typography Matters

Typography is the art and technique of arranging type to make written language legible, readable, and visually appealing when displayed.

Type is not just letters — it carries tone, personality, authority, and emotion. The right typeface communicates before a single word is read.

🎓 Classifications

4 Typeface Classifications

Serif

Serif

Small strokes at letter ends. Traditional, authoritative, trustworthy.

e.g. Times New Roman, Garamond, Georgia

Use: formal documents, books, newspapers

Sans

Sans-Serif

No strokes at ends. Modern, clean, accessible.

e.g. Helvetica, Arial, Futura, Gill Sans

Use: screens, signage, modern design

Mono

Monospaced

Each character occupies equal width. Technical, precise.

e.g. Courier, Space Mono, Consolas

Use: code, forms, typewriter aesthetic

Script

Script / Decorative

Mimics handwriting or has unique styling. Expressive, personal.

e.g. Brush Script, Comic Sans, Display fonts

Use: invitations, branding, titles (not body)

Legibility tip

Serif and sans-serif are most legible for body text. Script/decorative should be used sparingly — for display headings only, never for long text passages.

🎓 Hierarchy

Typographic Hierarchy

Typographic hierarchy is the arrangement of type to guide the viewer's reading order by signalling what is most important.

Creating Hierarchy Through

  • Size — larger = more important
  • Weight — bold draws the eye first
  • Colour/Contrast — saturated vs muted
  • Position — top/centre = primary
  • Typeface — serif heading + sans body
  • Spacing — isolation = importance
Exam Sentence Starter

"Typographic hierarchy is established through the use of [larger/bolder/coloured] type for the [heading], which signals to the viewer that this information has greater priority than the [body text/caption]."

HEADLINE
Subheading establishes context
Body text provides detailed information for the reader. It is smaller and lower contrast so the eye processes it after the headline and subheading have established context.
Caption — smallest, lowest priority
① Headline → ② Sub → ③ Body → ④ Caption
🎓 Spacing

Spacing: Kerning, Tracking, Leading

Kerning

AV
AV

Adjustment of space between two specific characters. Corrects optically uneven pairs like AV, WA, To, Ty.

Without kerning: A  V looks wrong
With kerning: AV looks right

Tracking

TRACKING
TRACKING

Adjustment of space across a range of characters (whole word, line or paragraph). Affects the overall density and mood of text.

Loose tracking → airy, luxury feel
Tight tracking → dense, urgent feel

Leading

Line one tight
Line two tight
Line one loose
Line two loose

The vertical space between lines of text. Named after the lead strips used in manual typesetting. Affects readability of body text.

Memory aid

Kerning = two characters · Tracking = a range of text · Leading = line spacing (vertical)

🎓 Type as Element

Type as a Design Element

In VCD, type is not just content — it is a visual element that contributes line, shape, tone, texture, and rhythm to a design.

Type Contributes to Elements

  • Line — strokes and stems of letterforms
  • Shape — the silhouette of type blocks
  • Tone — weight variations (light to black)
  • Texture — body text creates a tonal texture
  • Point — a single character as a focal mark
Exam Sentence Starter

"The [serif/sans-serif] typeface used for the heading contributes [visual weight/authority/elegance] to the design, while the [classification] body text ensures legibility and creates a [tonal texture/visual rhythm]."

🎓 Type in Context

Choosing the Right Typeface

Serif — When to use

  • Formal, traditional, authoritative tone
  • Print body text (easier to read in print)
  • Law firms, banks, newspapers, books
  • When you want to convey heritage or trust

Sans-Serif — When to use

  • Modern, clean, accessible designs
  • Screen interfaces and apps
  • Signage and wayfinding
  • Tech companies, startups, modern brands

Script/Decorative — Caution

  • Headlines and display only — never body text
  • Invitations, greeting cards, branding
  • Can feel unprofessional if overused
  • Avoid for any text under 14pt

🎓 Layout

Type and the Grid

A typographic grid is an invisible framework of columns, rows, and margins that organises type and images on a page.

Grid Components

  • Margins — blank space around the page edge
  • Columns — vertical divisions for text
  • Gutters — space between columns
  • Baseline grid — horizontal lines for text alignment
  • Flowlines — horizontal alignment guides
A 3-column grid allows a headline to span all 3 columns, while body text sits in one column and an image fills two — creating varied, dynamic layouts.
📝 Exam Technique

Writing About Typography

✅ Strong Response

"The serif typeface used for the masthead communicates authority and tradition, consistent with the newspaper's long-established identity. The larger point size and bold weight create typographic hierarchy, establishing the masthead as the primary visual element. The body text in a smaller sans-serif typeface ensures legibility at smaller sizes while providing visual contrast with the heading."

❌ Weak Response

"The font is big and bold. It uses a nice serif font for the title."
